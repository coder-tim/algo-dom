AlgoDOM.js
About (current) Documentation

Getting Started


  1. Download the code from here.

  2. In your web application's .html file, include algodom.js using the following tag: 
                         
              <script type="text/javascript" src='algodom.js'></script>



API Documentation

1. Array


  • Class Instantiation
                           
                let exampleArray = new AlgoDomArray(color, dataStructureContent, uiToggle)


  • Parameters
    Parameter Name Type Description Example
    color string The color of the data structure that will be generated. 'orange'
    dataStructureContent list of numbers or list of strings The content that will be contained in the data structure. [1, 3, 5, 7]
    uiToggle boolean This toggle is used to show/hide a UI that enables users to interact with the generated data structure diagram. true


  • Class Method(s) 
                     
                exampleArray.drawArray()


  • Returned Value
    • After you instantiate the class, the class method .draw() will return a div that contains the generated diagram.
    • You can append the returned div to your web app.


  • Example
                     
                // arrayContainer is a div in your web app
                const arrayContainer = document.querySelector("#arrayContainer");
                
                // Instantiate the class and call the method .draw()
                let exampleArray = new AlgoDomArray('green', [1, 3, 5, 6, 9, 1], true).drawArray();
                
                // Append the returned div into your web app's arrayContainer div
                arrayContainer.appendChild(exampleArray);

5. Animation


  • Class Instantiation
                         
              let algoDomAnimation = new AlgoDomAnimation(animationContainer, [frame1, frame2, frame3], animationSpeed)

  • Parameters
    Parameter Name Type Description Example
    animationContainer HTML div object A div object from your web app. animationContainer1
    [frame1, frame2, ...] List of objects Each object is a div that contains a data structure diagram. You can obtain such object by creating a static diagram using one of the methods from above (array, linked list, stack, queue). [linkedList1, linkedList2, linkedList3]
    speed A number in the range 1-3 This number indicates the animation speed. 1=slow, 2=medium, 3=fast 3


  • Class Method(s) 
                   
              algoDomAnimation.animate()


  • Example
                   
              const animationContainer1 = document.querySelector("#animationContainer1")

              // When you instantiate the AlgoDomLinkedList class, ensure that the 3rd parameter is set to false.
              // This will hide the interactive UI during animation, since the interactive UI is used for static diagrams only.
              let linkedList1 = new AlgoDomLinkedList('red', [1], false).drawLinkedList()
              let linkedList2 = new AlgoDomLinkedList('red', [1, 2], false).drawLinkedList()
              let linkedList3 = new AlgoDomLinkedList('red', [1, 2, 3], false).drawLinkedList()
              
              let algoDomAnimation = new AlgoDomAnimation(animationContainer1, [linkedList1, linkedList2, linkedList3], 3)
              algoDomAnimation.animate();
