Animations


Inserting elements into a linked list:
                     
               let linkedList1 = new AlgoDomLinkedList('red', [1]).drawLinkedList()
               let linkedList2 = new AlgoDomLinkedList('red', [1, 2]).drawLinkedList()
               let linkedList3 = new AlgoDomLinkedList('red', [1, 2, 3]).drawLinkedList()
               
               const animationContainer1 = document.querySelector("#animationContainer1")
               let algoDomAnimation = new AlgoDomAnimation(animationContainer1, [linkedList1, linkedList2, linkedList3], 3)
               algoDomAnimation.animate();
